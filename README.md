# DSA Problem Tracker (Python)

A command-line Python application to track Data Structures and Algorithms (DSA) problems.

## Features
- Add DSA problems with topic and difficulty
- Mark problems as solved
- View problems by topic
- Performance analysis of solved vs unsolved problems

## Data Structures Used
- Lists for storing problems
- Hash Maps (Dictionaries) for fast lookup and grouping

## How to Run
```bash
python3 tracker.py
